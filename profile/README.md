🙋‍♀️ 通过翻译英语技术文档的方式，学习第一手的技术文档，同时提高自己的英语水平。

🌈 主要翻译网络系统管理相关的技术文档，比如Linux, Windows, Network, Python, Ansible等官方文档或精选的技术文档。  

👩‍💻 内容主要由在校的学生完成，由专业技术的老师和英语老师审核。  

🍿 可能会有错误，请批评指正。
