## 参与贡献

1. Fork 您要翻译的技术文档仓库
2. 创建您的特性分支 (`git checkout -b feature/AmazingFeature`)
3. 提交您的更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到远程分支 (`git push origin feature/AmazingFeature`)
5. 打开一个 Pull Request
6. 等待审核
7. 审核通过后，合并到主分支
8. 完成
